This project showcases a machine learning approach to predicting survival in a simulated zombie apocalypse scenario. I used a k-Nearest Neighbors (k-NN) model to classify individuals as "human" or "zombie" based on factors like resource availability, environmental conditions, and proximity to urban areas. This project demonstrates how machine learning can estimate fish weight without using a physical scale. I built a k-Nearest Neighbors (k-NN) regression model using fish measurements like length, height, and width, along with species type, to predict weight. The workflow covers data collection, preprocessing, model tuning, and evaluation, achieving around 97–98% prediction accuracy with an optimal k value of 5.
